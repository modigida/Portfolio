Recipes

This is a WPF application designed as a digital recipe book. The app allows users to manage recipes and ingredients while utilizing a SQL Server database with Entity Framework for data management.

Key Features:

  • Recipe Management: Add, edit, delete and view detailed recipes.
  • Sorting Options: Sort recipes by preparation time, favorites, or name.
  • Ingredient Management: Manage a list of ingredients stored in the database.
  • User-Friendly Design: Intuitive interface built using the MVVM design pattern.

Technologies:

  • WPF (Windows Presentation Foundation)
  • C#
  • XAML
  • Entity Framework
  • SQL Server
  • MVVM
